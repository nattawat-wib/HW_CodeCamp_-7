2.1
property และ value คืออะไรแตกต่างกันอย่างไร 		Property คือ หัวข้อของคุณลักษณะที่เราต้องการทำการเปลี่ยนแปลง
Value คือ ปริมาณหรือสัดส่วนของคุณลักษณะที่เราต้องการเปลี่ยนแปลง
2.2
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง 		มี 10 ประเภท

1. Text
2. Font
3. List
4. Border
5. Table
6. Image
7. Background
8. padding
9. Margin
10. Display
2.3
value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร 		มี 2 ประเภท ใหญ่ๆ

• Absolute Length เป็นหน่วยวัดคงที่ ไม่อิงกับ element หรือแวดล้อมด้านข้างใดๆ
mm - มิลลิเมตร
cm - เซนติเมตร
in - นิ้ว
px - พิกเซล 1-96 นิ้ว
pt - พอยท์ 1/72 นิ้ว
pc - พิคาล 1/12 นิ้ว

• Relative Length จะไม่มีปริมาตรแน่นอน อาศัยการเทียบเคียงหรืออ้างจากแวดล้อม (หน่วยวัดเหล่านี้มีส่วนช่วยในการทำ responsive)
em - เท่ากับขนาด font ใน element เช่น กล่องข้อความจะความสูงเท่ากับ font
ex – เท่ากับความสูงของ font ใน element
ch – เท่ากับความกว้างของจำนวนตัวหนังสือใน parent element
rem – เท่ากับขนาด font ของ root element
vh – เท่ากับเปอร์เซ็นต์ความสูงของหน้าจอ
vw – เท่ากับเปอร์เซ็นต์ความกว้างของหน้าจอ
vmin – เท่ากับเปอร์เซ็นต์ของ view point ต่ำสุด
vmax - เท่ากับเปอร์เซ็นต์ของ view point สูงสุด
% - เท่ากับเปอร์เซ็นต์ชอง parent element
